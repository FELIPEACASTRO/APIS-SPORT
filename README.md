# APIS-SPORT

Aplicação Node.js para APIs de esportes.

## Instalação

```bash
npm install
```

## Executar

```bash
npm start
```

## Desenvolvimento

```bash
npm run dev
```

## Endpoints

| Método | Rota      | Descrição                  |
|--------|-----------|----------------------------|
| GET    | /         | Informações da API         |
| GET    | /health   | Status de saúde do servidor|
