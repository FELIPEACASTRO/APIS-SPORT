## Como usar este catálogo

1. **Para escolher uma API:** filtre por subcategoria e preço acima
2. **Para fazer a chamada:** use o `host` da terceira coluna como `X-RapidAPI-Host`
3. **JSON estruturado:** veja `catalog.json` + `catalog-part2.json` + `catalog-part3.json`
4. **Templates de código:** veja `templates/` (cURL, Node.js, Python)

> ⚠ APIs marcadas com `DESCONT.` foram descontinuadas e redirecionam para versões novas.

> 🎯 **Recomendação para começar:** as APIs com popularidade ≥ 9.7 e success_rate ≥ 95% são as mais confiáveis. Em particular: `Tank01` (MLB/NFL/NHL), `TheRundown`, `Sportspage Feeds`, `Pinnacle Odds API`, `PinBook Odds`, `Live Sports Odds (theoddsapi)` e `Bet365 API Inplay`.
